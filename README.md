# Interactive-Chaos-Theory-Simulator

ChaosLab — Interactive Chaos Theory Simulator
An interactive scientific visualization tool for exploring chaotic dynamical systems, strange attractors, and nonlinear behavior in real time.
Built using Python, Tkinter, Matplotlib, and SciPy, this simulator allows users to experiment with parameters and observe how deterministic systems produce complex, unpredictable behavior.

Features
Multiple Chaotic Systems
    Lorenz, Rössler, Chen, Aizawa, Thomas
    Dadras, Halvorsen, Bouali, Three-scroll attractor
    Hénon, Ikeda, Tinkerbell maps
    Clifford & De Jong attractors
    Logistic map (bifurcation diagram)

Interactive Parameter Control
    Real-time sliders for system parameters
    Instant visualization updates

Dynamic 3D Visualization
    Smooth rotation and animation
    Mouse-driven camera control
    Zoom & perspective interaction

Advanced Rendering
    Multiple colormaps
    Density-based visualization for fractal systems
    High-resolution plotting

Optimized Computation

Uses scipy.integrate.odeint for solving ODEs

Threaded rendering to keep UI responsive
